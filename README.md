# BalmJS template - `solid`

> A Solid project

## Requirements

```sh
yarn global add balm-core@next balm-cli
```

```sh
# Verify
balm -V

# Output:
# balm-cli: 2.0.0
# balm-core: 3.0.0
```

## Installation

```sh
balm init solid my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```
