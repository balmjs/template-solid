# BalmJS template - `solid`

> A Solid project

## Requirements

```sh
yarn global add balm-core@next balm-cli@next
```

```sh
# Verify
balm -V

# Output:
# balm-cli: 3.0.1
# balm-core: 4.5.5
```

## Installation

```sh
balm init solid my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```
